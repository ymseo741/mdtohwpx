# 연구개발계획서

## 과제 개요 및 개발 목표
**과제명**: 지능형 해상 충돌 방지 시스템(I-SCAS) 고도화  
**신청 기업**: (주)안티그래비티 | **기간**: 2026.03 ~ 2027.02

**핵심 개발 목표**:  
- AI 기반 충돌 위험 객체 식별 정확도 95% → 99% 달성
- 다중 선박 실시간 경로 예측 오차 범위 5m 이내 확보
- 엣지 컴퓨팅 기반 온디바이스 AI 추론 속도 150ms 이하 최적화

## 개발 방법 및 추진 전략

**핵심 기술 구성**:  
```
- 기술1: Transformer 기반 시계열 선박 궤적 예측 알고리즘
- 기술2: 고해상도 LiDAR 및 열화상 카메라 센서 퓨전 하드웨어
- 기술3: NMEA 2000 및 AIS 연동 실시간 데이터 버스 프로토콜
```

**개발 방법론**:  
```
1. 개발 환경 구축: GPU 가속 기반 딥러닝 학습 파이프라인 설계
2. 필드 테스트: 실제 선박 탑재를 통한 해상 환경 데이터셋 확보 및 미세 조정
3. 안정성 검증: 공인 시험 기관을 통한 기술 성능 평가지표(KPI) 달성 여부 검증
```

**추진 차별화 전략**:  
```
- 기존 선박 레이더 대비 악천후 시 탐지 거리 3.0배 개선
- 앤드-투-앤드(End-to-End) 학습을 통한 복잡한 해상 조우 상황 대응력 확보
- 위험관리: 시스템 장애 시 Fail-safe 기반 수동 모드 즉시 전환 대응책 마련
```
